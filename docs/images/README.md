# Screenshot Guide

Use demo data only. Do not include real names, real email addresses, private job
applications or personal documents.

Recommended resolution for all screenshots: 1440 × 900 or another 16:10 / 16:9
desktop format.

## dashboard.png

Show the main dashboard after demo login. The screenshot should include visible
summary cards and charts with realistic demo data.

## kanban-board.png

Show the Kanban board with several application cards across different columns.
Use prepared demo applications and avoid exposing private notes.

## application-details.png

Show a single application detail page with company, status and activity
information. Use demo data only.

## job-analysis.png

Show the job description analysis page with a sample analysis result. Do not use
a real job posting that contains private notes or personal data.

## resume-builder.png

Show the resume builder page with sample/demo content. Do not include a real
resume, real phone number, private address or real email address.
