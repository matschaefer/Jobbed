# Project Notes and Limitations

Status: Deployed portfolio project  
Last updated: July 2026

This file keeps a short list of practical limitations and follow-up ideas.

## Current Limitations

- The public demo is read-only.
- Real email delivery requires an SMTP provider.
- External AI output requires backend configuration and an API key.
- Uploaded documents are stored outside the Git repository.
- The application is focused on individual job seekers, not teams.

## Possible Future Improvements

- Add more end-to-end coverage for edge cases.
- Improve mobile layouts for dense application data.
- Add more export options for resumes and reports.
- Add optional cloud storage for uploaded documents.
- Add more detailed user profile matching for job analysis.

## Development Notes

The current setup is intentionally kept understandable for a junior fullstack
portfolio project. The focus is on a working Angular frontend, Spring Boot REST
API, PostgreSQL database, tests, Docker setup and deployed demo.
